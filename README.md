# ILOS-castillano
Integrated livestock Ordering System
Castillano’s Backyard: Integrated Livestock Ordering System (ILOS) is a web-based e-commerce platform that helps automate livestock ordering, manage inventory, and track sales for a small-scale backyard business. The system swaps out manual processes for a digital solution that boosts accuracy, efficiency, and decision-making through real-time monitoring and reporting.

### How the System Works
#### Customer Side
- Customer visits the website
- Browses available livestock products
- Adds items to cart
- Proceeds to checkout (Cash on Delivery only)
- Customer tracks order status
#### Admin Sode
- Admin logs into the system  
- Views dashboard summary  
- Manages products and updates inventory  
- Reviews and approves customer orders  
- System records sales automatically  
- Admin monitors reports and system activity
